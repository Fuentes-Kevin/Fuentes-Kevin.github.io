Kitchen
Timer

I began to work on this project because I wanted to test the skills
I learned and thought it would be fun to make something usefull my parents
can use at home.

The project involved hardware and software development, the use various technologies such as microcontrollers, WiFi modules, LED displays, and buzzer circuits. Extensive testing and optimization was conducted to ensure the timer's performance and reliable functionality in real-world kitchen scenarios.

The goal was to design and create a kitchen timer that can display a countdown and notify the user when the timer expires. It needs to include two buttons, one to start and stop the time, and another to increment the timer. Status LED's are needed to show if the timer is counting down. A buzzer is implemented to sound an alarm. Lastly it needs to be able to communicate with a smart phone using WiFi.

I plan to revisit the project in order to create a housing for the PCB board to ensure the components are protected while still allowing the user to interact with the device. At the same time it will give the device a cleaner and more user-friendly look.
